# Debug Mastery Frontend

This is a starter code for the frontend of the Debug Mastery project using React.

## Structure
- `src/`: Contains application code.
- `public/`: Static assets.
- `.env`: Configuration file for environment variables.
- `package.json`: Project dependencies and scripts.

## Sample React Code
```javascript
import React from 'react';

function App() {
  return (
    <div className="App">
      <h1>Debug Mastery Frontend</h1>
    </div>
  );
}

export default App;
```
